To create the Ahk2Exe launcher:

1) re-name appinfo.ini to appinfo.iniX  (any name will do, as long as it's changed)

2) re-name appinfo_Ahk2Exe.ini to appinfo.ini

3) re-name Launcher/Custom.nsh to Custom.nshX  (any name will do, as long as it's changed)

4) Run PortableApps.com Launcher, it should generate a file name AutoHotkey2ExePortable.exe in the App's root directory

5) re-name files from step 1-3 back to their original names
